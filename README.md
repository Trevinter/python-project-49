## Requirements:

Python 3.10 or above

## Description:

### Brain-games - the package contains five simple mathematics quiz games:

1 brain-even;
2 brain-calc;
3 brain-gcd;
4 brain-progression;
5 brain-prime.

## How to install:

### You need to write step-by-step the following commands in your terminal:

make install

make build

make package-install
